## Тест-кейсы

### 1) Успешный вход по номеру телефона и OTP-коду
Предусловия:
- Пользователь с существующим Яндекс.ID зарегистрирован (известен номер телефона и почта).
- Пользователь авторизован в мобильном приложении Яндекс.Почта.
- У пользователя есть доступ к мобильному устройству с приложением.

Тестовые данные: Номер телефона пользователя.
| Шаги: |Ожидаемый результат:  |
|--|--|
|1. Открыть мобильный браузер и перейти на страницу авторизации (passport.yandex.ru), нажать на вкладку "Телефон".|Открылась страница с полем ввода для номера телефона и кнопкой "Далее". |
|2. В появившемся поле ввода ввести номер телефона пользователя и нажать "Далее".|Появилось новое поле ввода для OTP-кода и кнопка "Отправить повторно". В мобильное приложение Яндекс.Почта пользователя отправился OTP-код для авторизации. |
|3. Открыть мобильное приложение, скопировать код и вставить в поле ввода на passport.yandex.ru.|OTP-код принят системой. |
|4. Нажать кнопку подтверждения входа.|Пользователь успешно авторизован и перенаправлен в аккаунт. |
